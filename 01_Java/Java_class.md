# 클래스
- 정의된 내용을 메모리에 실체화 시키는 행동
- 자료형의 역할을 함 > 변수를 만들 수 있음
- 데이터 저장의 구조 생성 > 관리 수월(용이) > 가독성 향상
- 각 요소의 이름 부여 > 각각의 데이터가 어떤 데이터인지 식별이 용이
- 요소의 이름이 동일함 <br><br>

## 클래스 사용
### 1. 클래스 선언하기(정의)
- 클래스명 > 파스칼 표기법
```java
class 클래스명 {
  클래스 멤버 구현
    a. 멤버 변수
    b. 멤버 메소드
}
```
### 2. 클래스 사용 -> 객체 생성
```java
클래스명 객체명 = new 생성자();
```
